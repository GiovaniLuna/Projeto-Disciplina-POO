# Bus-ticket-manager
Um projeto em grupo da disciplina de Programação Orientada a Objetos da Universidade Federal de Sergipe.

Para rodar:
```
python main.py
```

- Cadastro de viagens

- Manutenção de viagens

- Venda de passagens

- Troca de passagens
